# projects

Fetches projects from ElevenLabs.

## Example Usage

```hcl
data "projects" "example" {
}
```

## Argument Reference

- No configurable arguments.

## Attribute Reference

- `projects` - Computed by the API.
- `id` - Computed by the API.
- `name` - Computed by the API.
- `state` - Computed by the API.
