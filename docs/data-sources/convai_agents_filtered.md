# convai_agents_filtered

Fetches convai agents filtered from ElevenLabs.

## Example Usage

```hcl
data "convai_agents_filtered" "example" {
}
```

## Argument Reference

- No configurable arguments.

## Attribute Reference

- `agents` - Computed by the API.
- `id` - Computed by the API.
- `name` - Computed by the API.
- `created_at` - Computed by the API.
- `updated_at` - Computed by the API.
