# pronunciation_dictionaries

Fetches pronunciation dictionaries from ElevenLabs.

## Example Usage

```hcl
data "pronunciation_dictionaries" "example" {
}
```

## Argument Reference

- No configurable arguments.

## Attribute Reference

- `dictionaries` - Computed by the API.
- `id` - Computed by the API.
- `name` - Computed by the API.
