# voices

Fetches voices from ElevenLabs.

## Example Usage

```hcl
data "voices" "example" {
}
```

## Argument Reference

- No configurable arguments.

## Attribute Reference

- `voices` - Computed by the API.
- `id` - Computed by the API.
- `name` - Computed by the API.
- `category` - Computed by the API.
