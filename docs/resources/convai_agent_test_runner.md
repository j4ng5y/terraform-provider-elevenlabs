# convai_agent_test_runner

Manages convai agent test runner in ElevenLabs.

## Example Usage

```hcl
resource "convai_agent_test_runner" "example" {
}
```

## Argument Reference

- No configurable arguments.

## Attribute Reference

- `results` - Computed by the API.
- `test_id` - Computed by the API.
- `status` - Computed by the API.
- `passed` - Computed by the API.
- `message` - Computed by the API.
- `duration_ms` - Computed by the API.
- `started_at` - Computed by the API.
- `completed_at` - Computed by the API.
